# Frequently Asked Questions (FAQ)

## ❓ General
**Q: What is cntm-graph?**
A: High-performance, low-level graph engine for AI memory mapping.

## 🛠️ Technical
**Q: How do I run tests?**
A: Use the command: `cargo test`.

**Q: Which languages are supported?**
A: Primarily `Rust, Shared Memory (SHM), FlatBuffers, SIMD (AVX-512), Mojo FFI`.
